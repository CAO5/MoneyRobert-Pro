## 三、学习路径建议

### 路径 A：标准路径（推荐）

```
01 小白 → 02 熟练 → 03 资深 → 04 设计者
```

每篇通过结业测试后再进入下一篇，预计每篇学习 + 实操 1-2 周。

### 路径 B：已有基础快速路径

如果你已经熟悉 Docker / Rust / Vue / 加密交易，可以直接：

- 跳过 01 篇，从 02 篇开始
- 02 篇通过结业测试后，03 篇重点学习风控调优与晋级门禁
- 04 篇按需查阅（哪一章要改就看哪章）

### 路径 C：纯运维路径

只部署不深度使用：

- 01 篇必学
- 02 篇第 8 章（紧急停止与日常维护）+ 03 篇第 10 章（数据库运维）

## 五、系统核心能力速览

| 能力域 | 系统提供 | 用户需要做 |
|--------|---------|----------|
| 行情采集 | 8 个并行任务自动采集 OKX 公开数据 | 验证数据质量 |
| 多 Agent 辩论 | 13 类 Agent 4 阶段辩论 + 基金经理聚合 | 启动辩论、解读结果 |
| 概率决策卡 | 输出概率分布 + EV + CVaR + 失效条件 + 信任等级 | 填表生成、按门禁执行 |
| 回测验证 | 三级可信等级门禁 | 创建任务、等待结果、按门禁判定 |
| 模拟盘自动交易 | paper/demo/live 三模式 + 5 分钟自动循环 | 配置参数、启动、监控 |
| 风控执行 | 单笔 + 组合双层风控 | 设置合理阈值 |
| 晋级流程 | Level 0→1→2→3 三步审核（initiate→observe→approve） | 满足条件后人工审批 |
| 紧急停止 | emergency/stop API | 异常时立即触发 |
