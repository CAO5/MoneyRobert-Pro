# MoneyRobert 虚拟货币低成本出海方案

> 版本日期：2026-06-29  
> 核心目标：非托管虚拟货币系统交易、无强制费用，通过独立自愿赞助支持产品，并将首年出海现金成本压到最低。  
> 本方案不替代目标法域律师、税务师或监管机构的书面意见。
>
> **收费更新：** 文中原有“10% 盈利服务费”及分账测算已经废止，最新规则以[《MoneyRobert 自愿赞助模式与监管边界》](./MoneyRobert自愿赞助模式与监管边界.md)为准。改变收费名称不会消除系统交易本身的许可义务。

## 一、结论

放弃早期采用 ADGM + Cayman 的重型结构，改为：

### 阶段一：持牌合作 MVP

`MoneyRobert Mauritius GBC（海外技术和利润主体） → 毛里求斯现有持牌 AI/资产管理机构 → 客户本人交易所账户`

- 持牌合作方作为受监管服务提供者，与客户签署管理或顾问协议。
- MoneyRobert 提供算法、系统执行、订单归因和高水位线账本。
- 客户资产始终留在客户本人交易所账户。
- 合作方收取 10% 盈利服务费，再按合同向 MoneyRobert 支付技术和策略收入分成。
- 不设 Cayman、不设 ADGM、不设海外基金、不自建托管。

目标成本：**首年 USD 25,000–60,000**，预计 2–4 个月进入封闭测试。具体取决于持牌合作方报价、法律意见和当地人员安排。

### 阶段二：自有牌照

当系统管理账户达到 USD 5–10 million、连续两个季度产生可核验盈利后，申请：

- Mauritius Global Business Licence。
- FSC `FS-1.18 Robotic and Artificial Intelligence Enabled Advisory Services`。
- 根据 FSC 对具体交易链路的书面预沟通，决定是否叠加 `FS-1.1 Assets Management` 或其他许可。

自有牌照阶段首年预算：**USD 60,000–120,000**，另维持 MUR 600,000 最低未受损资本。

## 二、为什么选择毛里求斯

毛里求斯 FSC 已提供专门的机器人和 AI 投资顾问许可：

- 官方申请费：USD 500。
- 官方固定年费：USD 1,900。
- 最低未受损资本：MUR 600,000。
- 牌照可以通过算法提供投资建议，并在证券范围内管理客户组合。
- 董事会至少三人，其中一名为独立董事及毛里求斯居民。
- 主要银行账户设在毛里求斯，并配置与业务规模匹配的人员、风控和网络安全控制。

FSC 同时明确认可虚拟资产可作为专业和专家投资者的投资资产类别。因此，毛里求斯是目前兼具“AI 管理框架、虚拟资产监管基础、英语环境和较低官方费用”的方案。

但必须注意：虚拟资产是否完全落入 FS-1.18、是否需要额外资产管理或 VASP 权限，要由 FSC 按实际业务确认。VAITOS 的 Class I 主要针对代币发行和销售相关金融服务，并不是通用的虚拟货币投资顾问牌照，不能看见 “Advisory” 字样就直接申请。

## 三、最低成本公司结构

```text
创始股东
   │
   ▼
MoneyRobert Mauritius GBC
海外收入、IP、产品和合同主体
   │
   ├── 持牌合作阶段：向当地持牌机构提供技术与策略
   │
   └── 自有牌照阶段：直接成为受监管经营主体
   │
   ▼
毛里求斯持牌 AI/资产管理机构
客户准入、适当性、监管责任、收费复核
   │
   ▼
客户本人 CEX 账户
Trade-only API，无提现权限
```

早期只设一个海外公司：

- 不设 Cayman TopCo：融资前没有必要。
- 不设 IP HoldCo：避免转让定价、银行和实质经营成本。
- 不设基金：资金不归集，不承担基金行政、审计和托管成本。
- 不设自营交易公司：客户账户与公司自有资金完全分开。
- 不收 USDT：初期用银行或受监管支付机构收取法币账单，避免增加钱包、转账和 Travel Rule 范围。

如果短期还未能设立 GBC，可由持牌合作方承接全部客户和收费，MoneyRobert 仅作为境外技术供应商；但不能由中国境内个人或私人钱包收款。

## 四、必须收缩的产品范围

低成本不是选一个监管宽松地区，而是主动减少受监管活动。

### 允许范围

- 客户本人在第三方中心化交易所开户。
- API Key 只能读取和交易，禁止提现、转账和创建新 API。
- 首期只做 BTC/USDT、ETH/USDT 现货。
- 仅邀请专业、专家或符合当地规定的高净值客户。
- 系统自动执行预先授权的策略，客户可以立即暂停和撤销 API。
- 只按系统订单已实现净盈利收取 10% 服务费。

### 首期禁止

- 永续合约、期权、杠杆和借贷。
- DeFi 钱包签名、链上跨协议操作。
- 质押、理财、代币发行、跟单市场。
- 客户资产归集、统一钱包、内部账本余额。
- MoneyRobert 代客户充值、提现或转账。
- 向零售用户公开承诺自动赚钱。
- 用 USDT 私人钱包收取服务费。

现货 BTC/ETH + 非托管 CEX API 的价值在于：

- 不保管私钥。
- 不转移客户虚拟资产。
- 不经营交易所或撮合系统。
- 不发行代币。
- 容易获得完整成交和手续费记录。

这不会自动消除投资管理许可，但能显著减少托管、钱包、转账、Travel Rule、衍生品和市场运营许可。

## 五、地域策略

毛里求斯牌照不具备全球通行权，客户所在地规则仍然适用。

### MVP 准入

- 仅限经律师确认可以接受境外管理服务的专业/专家客户。
- 采用邀请制，不进行公开广告或应用商店大范围投放。
- 每个国家建立准入矩阵和法律意见编号。
- 注册时核验国籍、税务居民、常住地、IP、电话、银行和交易所 KYC 国家。

### 默认封锁

- 中国大陆。
- 美国及美国人士。
- 英国、欧盟、加拿大、澳大利亚、日本、新加坡、香港零售客户。
- 受制裁、FATF 高风险及交易所禁止地区。

首期不要写“全球可用”。正确文案是“仅向获准地区的专业客户开放”。

## 六、盈利收费模型

### 1. 计费公式

`系统净利润 = 系统平仓盈利 - 系统平仓亏损 - 手续费 - 资金成本 - 税费 - 历史待弥补亏损`

`服务费 = max(0, 超过账户高水位线的系统净利润 × 10%)`

虽然首期只做现货，公式仍保留未来资金费和融资费用字段。

### 2. 结算频率

- 每日计算虚拟账单。
- 每月向客户展示。
- 每季度结晶并收费。
- 客户关闭策略时补充结算。

季度结晶比每月收费更能避免短期盈利收费后立即回撤的争议。

### 3. 持牌合作期分账

建议谈判区间：

- 客户盈利服务费：10%。
- 持牌合作方：收费收入的 20%–40%。
- MoneyRobert：收费收入的 60%–80%。
- 交易所返佣必须向客户披露，并优先冲减交易成本，避免与增加交易频率形成利益冲突。

分账依据是服务收入，不是转走客户交易利润。客户先保留全部交易资产，再通过账单支付服务费。

## 七、交易所接入标准

只接入满足以下条件的交易所：

- 允许机构策略管理或第三方交易 API。
- API 权限可强制关闭提现和转账。
- 支持 IP 白名单、子账户和密钥轮换。
- 提供订单、成交、手续费、充值提现和账户余额完整历史。
- 具有明确服务地域及专业客户规则。
- 能提供机构支持和异常订单协助。

技术控制：

- API Secret 使用云 KMS/HSM 加密，数据库只保存密文。
- 交易服务与用户、账单、后台系统网络隔离。
- 单账户、单币种、单日损失和全局熔断。
- 行情延迟、订单拒绝、成交偏差和余额异常自动停机。
- 所有策略版本、信号、订单和人工操作可重放。
- 禁止任何服务端代码读取提现权限。

## 八、税务方案

毛里求斯一般公司所得税率为 15%。

毛里求斯税务局明确将 FSC 持牌的机器人与 AI 投资顾问服务收入列为可申请 80% 部分豁免的活动。满足以下实质条件时，理论有效税率可能为 3%：

- 核心收入活动在毛里求斯开展。
- 有足够的合格人员。
- 在当地发生与业务规模相称的支出。
- 取得并持续满足相关 FSC 许可。

商业预算先按 15% 税率计算；只有税务师确认收入分类与实质条件后，才在财务模型中采用 3%。不能只租地址和名义董事来申请部分豁免。

如果核心管理、策略审批、财务人事和客户服务仍全部在中国，仍存在中国实际管理机构、常设机构和转让定价风险。最低成本不等于零海外人员。

## 十、盈亏平衡

以 10% 盈利服务费、持牌合作方分走 30%、年度运营固定成本 USD 40,000 估算：

| 管理账户规模 | 系统年度已实现净收益率 | 毛服务费 | MoneyRobert 分成后收入 |
| ---: | ---: | ---: | ---: |
| USD 2m | 10% | USD 20,000 | USD 14,000 |
| USD 5m | 10% | USD 50,000 | USD 35,000 |
| USD 8m | 10% | USD 80,000 | USD 56,000 |
| USD 10m | 10% | USD 100,000 | USD 70,000 |

按该假设，合作模式约需 USD 6m–8m 盈利账户规模才可能覆盖运营成本。这里的 10% 系统收益率只是敏感性参数，不是收益目标或对外承诺。

自有牌照的切换条件：

- 可计费账户规模达到 USD 5m–10m。
- 连续两个季度产生已实现净盈利。
- 预计未来 12 个月向合作方支付的分成高于自有牌照增量成本。
- 系统、人员和合规已具备独立接受监管检查的能力。

## 十一、90 天最低成本执行计划

### 第 1–14 天

- 将产品范围锁死为 BTC/ETH 现货、CEX、非托管、专业客户。
- 获得毛里求斯律师对 FS-1.18 + 虚拟资产交易 API 的书面范围意见。
- 联系 3–5 家持有 FS-1.18、资产管理或适当许可的机构。
- 询问 FSC 预沟通：是否需要叠加 Assets Management 或其他许可。

### 第 15–30 天

- 确定持牌合作方和收入分成。
- 设立单一 Mauritius GBC；不设 Cayman。
- 确定一家交易所、一个策略和一个基础币种。
- 完成地域封锁与专业客户准入规则。

### 第 31–60 天

- 完成 trade-only API、IP 白名单、KMS 和全局熔断。
- 完成系统订单归因、高水位线和季度结晶。
- 完成英文合同、风险揭示、适当性和 KYC。
- 影子盘跑满 30 天，不收费。

### 第 61–90 天

- 邀请 10–20 名专业客户，低额度实盘。
- 前 30 天继续不收费，只验证成交和账单。
- 由合作方复核首份虚拟账单。
- 无严重交易、归因或安全事故后，开启盈利服务费。

## 十二、不能为了省钱做的事情

- 塞舌尔或 BVI 注册一个 IBC 就声称“全球合规”。塞舌尔 FSA 已明确：仅注册 IBC 不能提供虚拟资产服务，代客户交易或提供投资建议会进入监管范围。
- 把自动交易写成“软件工具”就不申请许可。监管看实际控制和订单权限，不只看合同名称。
- 使用客户主密钥、助记词或提现权限。
- 通过私人 USDT 钱包收取盈利分成。
- 接受中国大陆或美国用户使用 VPN 绕过地区限制。
- 为了提高收费频繁交易、隐藏返佣或只展示盈利账户。
- 将实盘策略部署在无法审计的个人服务器。
